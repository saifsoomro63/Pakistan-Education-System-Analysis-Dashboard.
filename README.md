# Pakistan-Education-System-Analysis-Dashboard.
This project features an interactive dashboard designed to analyze the landscape of the education system in Pakistan from 2022 to 2025. By utilizing Looker Studio, this dashboard consolidates complex school data into a clear, visual format to assist stakeholders in identifying resource gaps and educational trends.

# Key Insights & Dashboard Features: <b>
<b> **Geographic Distribution:** <b>
<b> An integrated map provides a high-level view of school density across various districts and provinces, enabling users to pinpoint geographic areas with lower school availability.  <b>
**Facility Access:** <b>
The dashboard evaluates the presence of critical infrastructure—including internet connectivity, clean water, and electricity—across different school types, such as Government, Private, and Madrasah systems.  <b>
**Ownership & Language:** <b>
Visualizations display the distribution of school ownership and mediums of instruction (e.g., Urdu, English, and regional languages), highlighting the diverse educational offerings across the country.<b>  
**Infrastructure Readiness:** <b>
Through detailed analysis of building conditions and area types (Urban, Rural, Semi-Urban), the dashboard helps identify schools that are in "Poor" or "Very Poor" condition, emphasizing the need for targeted maintenance.  <b>
**Contextual KPIs:** <b>
The dashboard tracks total student enrollment, teacher statistics, and annual budgetary data to provide a holistic view of the system’s current capacity.  <b>
# Technical Approach: <b>
The project employs advanced Data Studio features, including cross-filtering and calculated fields, to transform raw binary data into actionable, easy-to-read metrics. This ensures that users can drill down into specific regions or school types to extract nuanced insights.  <b>

This project serves as a demonstration of data storytelling, aimed at supporting evidence-based policy planning and efficient resource allocation across the Pakistani education sector.
